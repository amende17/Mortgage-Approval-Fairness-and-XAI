# Mortgage Approval Fairness and XAI

## Overview

This project explores the use of machine learning, fairness assessment, and explainable AI techniques to analyze mortgage approval decisions. Using a mortgage lending dataset, a predictive model was developed to identify approval outcomes while evaluating potential disparities across demographic groups.

The project focuses on building transparent and responsible AI systems by combining predictive modeling with fairness evaluation and model interpretability techniques.

## Objectives

* Predict mortgage approval outcomes using machine learning.
* Evaluate model performance using classification metrics.
* Assess fairness across demographic groups.
* Identify potential sources of bias in lending decisions.
* Improve model transparency through explainable AI techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Fairlearn
* SHAP
* LIME
* Matplotlib
* Seaborn

## Methodology

### Data Preparation

* Data cleaning and preprocessing
* Handling missing values
* Feature selection and transformation
* Training and testing dataset creation

### Predictive Modeling

* Logistic Regression model development
* Performance evaluation using classification metrics
* Analysis of approval prediction outcomes

### Fairness Assessment

* Group-level fairness evaluation using Fairlearn
* Comparison of approval rates across demographic groups
* Identification of disparities and potential allocation harms

### Explainable AI

* SHAP analysis for global feature importance
* LIME analysis for local prediction explanations
* Interpretation of factors influencing mortgage approval decisions

## Key Outcomes

* Developed an end-to-end mortgage approval prediction pipeline.
* Evaluated fairness metrics across demographic groups.
* Generated interpretable explanations for model predictions.
* Demonstrated the importance of transparency and fairness in financial decision-making systems.
